# 14 Future Work

## Scaling AeroDemand AI
While the current version of **AeroDemand AI** is a robust and functional prediction system, there are several avenues for future enhancement:

### 1. Advanced Machine Learning
- **Deep Learning**: Implementing LSTM (Long Short-Term Memory) networks for even better time-series forecasting.
- **Ensemble Methods**: Combining multiple models (XGBoost, ARIMA and Linear Regression) to reduce prediction error.

### 2. Expanded Feature Set
- **External Factors**: Integrating external data such as Jet Fuel prices, GDP growth rates, and global holidays.
- **Competitor Analysis**: Scraping or connecting to APIs to include competitor pricing in the prediction model.

### 3. Enterprise Features
- **User Authentication**: Implementing secure login for different levels of management access.
- **Scenario Archiving**: Allowing users to save and compare different "What-If" prediction scenarios.
- **Real-time Data Streams**: Connecting to live flight booking databases for "Next-Day" demand forecasting.

### 4. Advanced Visualization
- **Geospatial Mapping**: Visualizing demand by destination and origin on an interactive world map.
- **Automated Reporting**: Sending weekly PDF demand summaries to stakeholders via email.
